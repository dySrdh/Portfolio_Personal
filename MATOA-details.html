Matoa - Smart Home IoT System

Matoa is an innovative IoT solution designed to help homeowners monitor and control various aspects of their home environment remotely. The system utilizes Arduino-based hardware sensors to collect real-time data on temperature, humidity, light levels, and motion detection throughout the home.

The collected data is transmitted to a Node.js backend server and stored in MongoDB, allowing users to access comprehensive insights through an intuitive web dashboard. The system also features automated response capabilities, such as adjusting connected smart devices based on environmental changes or user-defined rules.

Key Features

  • Real-time environmental monitoring (temperature, humidity, air quality)
  • Motion detection and security alerts through mobile notifications
  • Energy consumption tracking and optimization suggestions
  • Automation rules engine for controlling connected devices
  • Historical data analysis and trend visualization
